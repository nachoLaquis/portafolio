TASKER: Sistema de gestión de tareas

  • Categoría: Full Stack (React + AppScript)
  • Cliente: Proyecto personal
  • Fecha: 2024

Descripción

TASKER es un sistema de gestión de tareas que aprovecha la simplicidad de Google Sheets como backend y la potencia de React JS en el frontend. Su objetivo es ofrecer una solución ligera, accesible y fácil de usar para docentes y usuarios que necesitan registrar, organizar y aprobar tareas sin complicaciones técnicas. El sistema integra formularios dinámicos, listas desplegables configurables y un CRUD completo sobre Google Sheets, todo dentro de una interfaz intuitiva construida con Ant Design y gestionada con React Router. Con este proyecto se busca mejorar la eficiencia en la gestión de actividades y facilitar la adopción de herramientas digitales en contextos diversos.

Sistema de Administración de Clientes y Apertura de Cuentas Bancarias

  • Categoría: Full Stack (React + Spring Boot + MySQL)
  • Cliente: Proyecto académico / Empresa financiera (caso práctico)
  • Fecha: 2024

Descripción

Proyecto full stack desarrollado para una empresa financiera, que integra un frontend en React con TypeScript y un backend en Spring Boot con base de datos MySQL, permitiendo registrar, modificar y eliminar clientes, así como gestionar la apertura y administración de cuentas bancarias. La solución ofrece formularios intuitivos y tablas dinámicas para visualizar clientes y sus cuentas, demostrando el dominio completo del ciclo de desarrollo desde la interfaz de usuario hasta la lógica de negocio y la persistencia de datos.

Generador de Informes Automatizados

  • Categoría: Google Apps Script / Automatización de reportes
  • Cliente: Enseña por Bolivia ExB
  • Fecha: 2024

Descripción

Proyecto web desarrollado con Google Apps Script que permite la generación automatizada de informes en Google Docs a partir de una plantilla en Google Sheets. La solución está pensada para facilitar el trabajo de los PEBS y docentes, quienes registran diariamente sus actividades en la hoja de cálculo y, al finalizar el mes, simplemente presionan “generar informe”. Esto ahorra tiempo y esfuerzo en la elaboración de reportes mensuales, optimizando la energía dedicada a tareas administrativas y permitiendo enfocarse en la labor educativa.

Tienda Online

  • Categoría: AppSheet / Gestión de ventas

Descripción

Sistema Web de Administración de Citas y Seguimiento de Tratamientos

  • Categoría: Desarrollo web
  • Cliente: Consultorio Dental “Real Dental”
  • Fecha: 2022

Descripción

Sistema web desarrollado para optimizar la administración de citas y el seguimiento de tratamientos odontológicos en el consultorio Real Dental. La solución fue construida con CodeIgniter 4 (PHP) y MySQL, aplicando la metodología ágil SCRUM. El sistema permite:

  • Programar, modificar y cancelar citas de manera rápida e intuitiva.
  • Registrar historias clínicas y diagnósticos mediante un odontograma digital.
  • Realizar seguimiento de tratamientos, incluyendo pagos y progreso de cada paciente.
  • Generar reportes en tiempo real sobre pacientes, citas, tratamientos y pagos.
  • Incorporar funcionalidades innovadoras como lectura de notificaciones por voz y dictado por voz para agilizar el llenado de información extensa.
Este proyecto responde a la necesidad de modernizar la gestión en consultorios odontológicos afectados por la pandemia, mejorando la atención al paciente y reduciendo el tiempo dedicado a tareas administrativas.
